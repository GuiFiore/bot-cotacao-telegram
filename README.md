# 🤖 Bot de Cotações — Telegram

Bot em Python que retorna cotações de Dólar, Euro e Bitcoin em tempo real.

## 🛠️ Tecnologias
- Python 3
- pyTelegramBotAPI
- AwesomeAPI (cotações gratuitas)

## ⚙️ Como usar
1. Clone o repositório
2. Instale as dependências: `pip install pyTelegramBotAPI requests`
3. Substitua o TOKEN no `bot.py`
4. Rode: `python bot.py`

## 💬 Comandos
- `/cotacao` — Dólar, Euro e Bitcoin agora
- `/ajuda` — Lista de comandos
